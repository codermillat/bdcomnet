# BDComNet — বাংলা গাইড
## বাংলাদেশ দুর্যোগ যোগাযোগ নেটওয়ার্ক

---

## BDComNet কী?

BDComNet হলো বাংলাদেশের জন্য একটি সৌরশক্তিচালিত, কমিউনিটি-পরিচালিত যোগাযোগ নেটওয়ার্ক।

**যখন সাইক্লোন আসে → মোবাইল টাওয়ার পড়ে যায়**
**যখন বন্যা হয় → ফাইবার ক্যাবল ছিঁড়ে যায়**
**যখন বিদ্যুৎ যায় → সব বন্ধ হয়ে যায়**

BDComNet এই সময়েও চালু থাকে। কারণ এটি কোনো কোম্পানি বা সরকারের উপর নির্ভরশীল নয়। এটি চলে সূর্যের আলোয় এবং রেডিও তরঙ্গে।

---

## এটি কীভাবে কাজ করে?

```
আপনার ফোন → WiFi সংযোগ → BDComNet টাওয়ার → বাংলাদেশের যেকোনো প্রান্ত
```

অথবা ছোট LoRa ডিভাইস দিয়ে:
```
আপনার LoRa নোড → রেডিও সংকেত → বন্ধুর নোড (১৫ কিমি পর্যন্ত)
```

---

## আপনি কী পাবেন?

| সুবিধা | বিবরণ |
|---|---|
| বিনামূল্যে মেসেজিং | WhatsApp-এর মতো, ইন্টারনেট ছাড়াই |
| বিনামূল্যে ভয়েস কল | পরিষ্কার কণ্ঠস্বর, সম্পূর্ণ বিনামূল্যে |
| GPS লোকেশন শেয়ারিং | পরিবারকে জানান আপনি কোথায় আছেন |
| গ্রুপ চ্যানেল | এলাকার সবার সাথে একসাথে যোগাযোগ |
| অফলাইন মানচিত্র | ইন্টারনেট ছাড়াই বাংলাদেশের মানচিত্র |

**সম্পূর্ণ বিনামূল্যে। চিরকালের জন্য। কোনো মাসিক বিল নেই।**

## কীভাবে যোগ দেবেন?

### পদ্ধতি ১ — WiFi টাওয়ার (সবচেয়ে সহজ, ৳০)
১. আপনার ফোনের WiFi চালু করুন
২. "BDComNet" নেটওয়ার্ক খুঁজুন
৩. সংযুক্ত হন — কোনো পাসওয়ার্ড নেই
৪. Element অ্যাপ খুলুন → শুরু করুন

### পদ্ধতি ২ — নিজের নোড বানান (৳২,৬৫০)
১. TTGO T-Beam ডিভাইস কিনুন (~৳১,৮০০)
২. Meshtastic অ্যাপ ইনস্টল করুন (বিনামূল্যে)
৩. ছোট সোলার প্যানেল লাগান
৪. ছাদে রাখুন → সারাজীবন চলবে

বিস্তারিত: [node-setup.md](node-setup.md)

## দুর্যোগের সময় কীভাবে ব্যবহার করবেন?

**বন্যার সময়:**
- আপনার লোকেশন পরিবারকে জানান
- উদ্ধারকারীদের সাথে যোগাযোগ করুন
- নিরাপদ জায়গার তথ্য শেয়ার করুন

**সাইক্লোনের সময়:**
- আশ্রয়কেন্দ্রের তথ্য পান
- পরিবারের সবার অবস্থান জানুন
- জরুরি সাহায্য চান

**ইন্টারনেট বন্ধ থাকলেও:**
- BDComNet চালু থাকে
- মেসেজ পাঠান, কল করুন
- সবার সাথে যুক্ত থাকুন
